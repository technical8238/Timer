# JavaScript Timer

A simple and responsive countdown timer built using HTML, CSS, and JavaScript. This project allows users to start, pause, resume, and reset the timer with an easy-to-use interface.

## Features
- ⏱️ Start Timer
- ⏸️ Pause Timer
- ▶️ Resume Timer
- 🔄 Reset Timer
- 📱 Responsive Design
- ⚡ Built with Vanilla JavaScript

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)

## Purpose
This project was created to improve my JavaScript skills, especially working with `setInterval()`, `clearInterval()`, DOM manipulation, and event handling.
